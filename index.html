  • Relative link type

same folder: To link to a file in the same folder,just use the file name.
(href = "reviews.html")

Child folder: For a child folder,use the name of child folder,followed by a
forward slash,then the file name.
(href = "music/listing.html")

Grand child folder: Use the name of the child folder, followed by a forward slash
then the name of the grand child folder, follwed by an other forward slash
then the file name.
(href = "movies/dvd/reviews.html")

Parent folder: Use ..// to indicate the folder above the current one,
then follow it with the file name.
(href = "..//index.html")

GrandParent folder: Repeat the ..// to indicate that you want to go up
two folders(rather then one),then follow it with the file name.
(href = "..//..//index.html")
